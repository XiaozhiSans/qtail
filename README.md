# qtail（QQ群名片小尾巴生成器）
  
这是一个用于生成qq群名片后面加个小尾巴的小工具  
  
# 注意
此生成器在鹅厂和谐了rlo和lro后仍然有效  
  
任何小尾巴在pcqq经典版上**都不会生效**
